Account & Balances API

Description of project 1.

Account and Balance API is a project that aims to simplify and centralize the process of gathering data related to accounts and balances in a financial services application. The API acts as an intermediary, consuming data from various downstream account and balance services and then aggregating this information. The aggregated data is made accessible to the application's presentation layer, which includes both the customer and employee portals.
